Things I Made

Some of my personal projects:

  • vesuvius
    Simple entity component system-based 3D game engine utilizing Vulkan API and Bullet Physics, written in C++. Made for my bachelor’s thesis.
  • Mercury
    Work-in-progress game engine using Google’s Filament as the renderer, Bullet for physics, LuaJIT for scripting, EnTT for entity component system management.
  • bevy_quickstart
    Project template for Bevy games.
  • World generator
    World map generator based on Amit Patel’s tutorial. Includes a 3D visualization of the world.
  • Core
    Programming language designed to be a stricter C with metaprogramming support. Built on top of LLVM.
  • extlang
    Simple extensible programming language based on the concept of an adaptive grammar, written in C++. It allows the user to alter its context-free grammar to suit their needs. Made for my master’s thesis.
  • noam
    C++ library that allows for creating lexers and parsers at runtime. Made for extlang.
